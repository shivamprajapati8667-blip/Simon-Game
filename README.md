# Simon Game

A memory-based game built using HTML, CSS, and JavaScript.

This project recreates the classic Simon Game where the player must remember and repeat a sequence of colors. With each correct round, the sequence becomes longer and more challenging.

---

## About the Project

The game generates a random color pattern. The player must repeat the pattern in the correct order. If the player makes a mistake, the game ends and can be restarted.

This project focuses on JavaScript logic, DOM manipulation, and event handling.

---

## Features

- Random sequence generation
- Increasing difficulty at each level
- Sound effects for each button
- Game over detection
- Restart functionality
